## ğŸ“ How to Customize

### 1. **Personal Information**

Open `index.html` and update the following:

```html
<!-- Email links -->
<a href="mailto:your.email@example.com">your.email@example.com</a>

<!-- GitHub links -->
<a href="https://github.com/yourusername">github.com/yourusername</a>

<!-- LinkedIn links -->
<a href="https://linkedin.com/in/yourprofile">linkedin.com/in/yourprofile</a>

<!-- College/University name -->
<p class="education-college">Add your college name</p>
```

### 2. **Update Projects**

Add or modify project cards in the Projects section:

```html
<div class="project-card">
    <div class="project-header">
        <h3>Your Project Name</h3>
        <span class="project-badge">Technology</span>
    </div>
    <p class="project-description">
        Your project description here...
    </p>
    <div class="project-tech">
        <span>Tech1</span>
        <span>Tech2</span>
    </div>
    <div class="project-links">
        <a href="#" class="project-link">View Project</a>
        <a href="#" class="project-link">GitHub</a>
    </div>
</div>
```

### 3. **Modify Skills**

Add or remove skills in the Skills section:

```html
<div class="skill-category">
    <h3>Your Skill Category</h3>
    <div class="skill-list">
        <span class="skill-tag">Skill1</span>
        <span class="skill-tag">Skill2</span>
    </div>
</div>
```

### 4. **Color Customization**

Edit the color variables in `styles.css`:

```css
:root {
    --primary-color: #2563eb;      /* Main color */
    --secondary-color: #1e40af;    /* Secondary color */
    --accent-color: #f97316;       /* Accent/highlight color */
    --text-dark: #1e293b;          /* Text color */
    --text-light: #64748b;         /* Light text color */
    --light-bg: #f8fafc;           /* Light background */
}
```

### 5. **Typography Changes**

Modify font sizes in `styles.css`:

```css
.hero-title {
    font-size: 3.5rem;  /* Change hero title size */
}

.section-title {
    font-size: 2.5rem;  /* Change section title size */
}
```
